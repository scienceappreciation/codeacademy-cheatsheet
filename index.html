HTML5 Cheatsheet

Semantic Elements

Element Description Usage
<main> Used to contain the dominant content of the page. <main>content</main>
<aside> Used to contain trivial or extra content. Usually paired alongside main in the body. <aside>content</aside>
<header> Commonly used to contain the title or nav of the page. <header>content</header>
<nav> Used to contain navigation links, commonly in an unordered list. <nav>links</nav>
<article> Represents a self-contained page of the document. Commonly paired with sections. <article>content</article>
<section> Represents a distinct part of an article. <section>content</section>
