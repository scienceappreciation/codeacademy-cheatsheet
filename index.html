HTML5 Cheatsheet

Semantic Elements

Element Description Usage
<main> Used to contain the dominant content of the page. <main>content</main>
<aside> Used to contain trivial or extra content. Usually paired alongside main in the body. <aside>content</aside>
<header> Commonly used to contain the title or nav of the page. <header>content</header>
<nav> Used to contain navigation links, commonly in an unordered list. <nav>links</nav>
<article> Represents a self-contained page of the document. Commonly paired with sections. <article>content</article>
<section> Represents a distinct part of an article. <section>content</section>

Media Embeds

Element Description Usage
<img> Used to put an image on the page. <img src="url/path">
<audio> Adds audio to the page with a player. <audio controls><source src="url/path"></audio>
<video> Adds video to the page with a player. <video controls src="url/path">Video not supported.</video>
<embed> Can be used for misc embeds such as GIFs. Legacy <embed type="MimeType" src="url/path">

Header Use

Element Description Usage
<h1> The highest section level. Usually reserved for the page title. <h1>text</h1>
<h2> The second highest section level. Use this to title sections. <h2>text</h2>
<h3> The third highest section level. Use this for parts of a section as needed. Go up the number order for sub parts up to 6. <h3>text</h3>

Quick Tables

Element Description Usage
<table> The first container used to create a table. <table>content</table>
<tr> Use inside of the table container to create a row. <tr>content</tr>
<td> Use inside of a table row to add a column/data. <td>data</td>
